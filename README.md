# 运势 (YunShi) - 命理全能 Skill

> 私人命理顾问 — 每日运程推送 · 八字紫微 · 占卜风水

[English](./README_EN.md) | 中文

---

## 功能特点

- **八字排盘**：四柱、日主、用神、神煞
- **紫微斗数**：命宫、十二宫、四化
- **奇门遁甲**：时辰局、起盘
- **梅花易数**：数字/时间起卦
- **六爻预测**：占卜问事
- **合婚分析**：双方八字匹配
- **风水布局**：财位、幸运颜色
- **每日运程**：早7点推送今日+晚8点推送明日

## 支持平台

- **OpenClaw** - 主要支持
- **Deeplumen** - 兼容

## 安装

```bash
# 方式1: 从源码安装
cd ~/.openclaw/workspace/skills
git clone https://github.com/你的用户名/yunshi.git

# 方式2: 使用 OpenClaw
openclaw skill install ./yunshi
```

## 环境要求

- Node.js >= 18
- 执行 `npm install` 安装依赖

## 快速开始

```
算命        → 八字排盘
今日运势    → 每日运程
2026年运势  → 流年分析
合婚 张三李四 → 合婚分析
```
